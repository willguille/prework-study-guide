HTML

  • The head element contains information about the webpage
  • The body element represent the visble content shown to the user
  • The section element define a section in a document that contains thematic content
  • The meta element represents metadata that cannot be expressed using the title, base, link, style, and script elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The text-align positioning property is used to center a title or heading on a webpage
  • The font-size property changes the size of the font
  • Always use link element on HTML file to link the CSS style sheet to the HTML

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls out the latest main branch
  • git push origin branch-name: push the changes we have locally to our remote GitHub branch
